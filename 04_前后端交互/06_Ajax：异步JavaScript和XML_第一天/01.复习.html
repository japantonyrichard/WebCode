1.获取前端表单提交的数据 1.1$_GET[] -- 获取get提交的数据 1.2$_POST[] -- 获取post提交的数据 2.php连接数据库 2.1创建数据库的连接 mysqli_connect('域名|ip','数据库用户名','数据库密码','数据库名称') 2.2设置编码格式，防止中文乱码 mysqli_query(数据库连接,'set names utf8') 2.3SQL语句 增删改查 2.4发送SQL mysqli_query(数据库连接,sql语句) 2.5关闭数据库--省略 mysqli_close(数据库连接) 3.mysql 3.1插入语句 -- 增 1）语法 insert into 表名 (字段1,字段2,...) values ('值1','值2',...) -- 插入指定的字段 insert into 表名 values (null,'值1','值2',...) -- 插入所有字段 2）返回值 返回布尔值 3.2删除语句 -- 删 1）语法 delete from 表名 where 条件 2）返回值 返回布尔值 3.3修改语句 -- 改 1）语法 update 表名 set 字段1='值1',字段2='值2' where 条件 2）返回值 返回布尔值 3.4查询语句 -- 查 1）语法 select 【信息】 from 表名 where 【条件】 2）返回值 返回对象 3）说明 ①【信息】 a、* -- 查询所有字段 b、指定字段名称 ②【条件】 a、1 -- 查询所有 b、并且 -- and c、或者 -- or
