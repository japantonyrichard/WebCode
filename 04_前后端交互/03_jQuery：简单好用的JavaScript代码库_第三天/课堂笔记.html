callback可以是匿名函数和有名函数的函数名 当同一个元素绑定了相同的事件不会发生覆盖 动态创建的元素没有事件 6.2事件绑定 1）bind() ①语法 JQ对象.bind(事件名称,callback); 当绑定一个事件： JQ对象.bind('事件名称',callback); 当绑定多个事件，处理程序相同时： JQ对象.bind('事件名称1 事件名称2',callback); 当绑定多个事件，处理程序不相同时： JQ对象.bind({ 事件名称1:function(){ }, 事件名称2:function(){ } }); ②说明 同一个元素绑定相同的事件，不会发生覆盖 动态创建的元素没有事件 2）on() ①语法 JQ对象.on(事件名称,callback); 当绑定一个事件： JQ对象.on('事件名称',callback); 当绑定多个事件，处理程序相同时： JQ对象.on('事件名称1 事件名称2',callback); 当绑定多个事件，处理程序不相同时： JQ对象.on({ 事件名称1:function(){ }, 事件名称2:function(){ } }); ②说明 同一个元素绑定相同的事件，不会发生覆盖 动态创建的元素没有事件 false可以作为一个函数的简写，一般用来阻止页面的提交或重置 例子：阻止页面的提交 $(':submit').on('click',false); ③语法 JQ对象.on(事件名称,选择器,callback) 说明： 选择器一般是JQ对象的后代 选择器是真正绑定事件的元素 选择器是一个字符串，例如选中li需要写成'li' 可以给动态创建的元素绑定事件 3）delegate() ①语法 JQ对象.delegate('选择器','事件名称',callback) ②说明 选择器一般是JQ对象的后代 选择器是一个字符串，例如选中li需要写成'li' 可以给动态创建的元素绑定事件 4）one() ①语法 JQ对象.one('事件名称',callback) ②说明 通过one绑定的事件只执行一次 其他语法同bind 6.3事件删除 1）unbind() 删除bind绑定的事件 JQ对象.unbind();//删除绑定的所有事件 JQ对象.unbind('事件1
